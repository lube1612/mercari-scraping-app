# Streamlitアプリ共有方法ガイド

このStreamlitアプリを他の人と共有する方法を説明します。

## 📋 共有方法の比較

| 方法 | 難易度 | 費用 | アクセス方法 | 推奨度 |
|------|--------|------|--------------|--------|
| Streamlit Cloud | ⭐⭐ | 無料 | URL共有 | ⭐⭐⭐⭐⭐ |
| ローカルネットワーク | ⭐ | 無料 | 同じWi-Fi内 | ⭐⭐⭐ |

## 🌐 方法1: Streamlit Cloudで公開（推奨）

### メリット
- ✅ 完全無料
- ✅ URLを共有するだけで使える
- ✅ 24時間自動で動く
- ✅ 世界中からアクセス可能

### デメリット
- ⚠️ Playwrightブラウザのインストールが必要（設定が必要）
- ⚠️ 無料プランには制限あり

### 手順

#### 1. GitHubにコードをアップロード

```bash
# GitHubでリポジトリを作成後
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/リポジトリ名.git
git push -u origin main
```

#### 2. Streamlit Cloudに接続

1. https://streamlit.io/cloud にアクセス
2. 「Sign up」→ GitHubアカウントでログイン
3. 「New app」をクリック
4. リポジトリを選択
5. Main file path: `streamlit_app.py` を指定
6. 「Deploy」をクリック

#### 3. URLを共有

デプロイが完了すると、以下のようなURLが生成されます：
```
https://your-app-name.streamlit.app
```

このURLを共有すれば、誰でも使えます！

### 必要なファイル

Streamlit Cloud用に、以下のファイルが必要です：

- `streamlit_app.py` - メインアプリ
- `requirements.txt` - 依存パッケージ（`requirements_streamlit.txt`をリネーム）
- その他の必要なPythonファイル

## 🏠 方法2: ローカルネットワークで共有

### メリット
- ✅ 簡単に設定できる
- ✅ 無料
- ✅ 同じWi-Fi内の人と共有可能

### デメリット
- ⚠️ あなたのパソコンが起動している必要がある
- ⚠️ 同じネットワーク内の人だけがアクセス可能

### 手順

#### 1. アプリを起動（外部アクセス許可）

```bash
streamlit run streamlit_app.py --server.address 0.0.0.0 --server.port 8501
```

#### 2. IPアドレスを確認

```bash
# macOSの場合
ifconfig | grep "inet " | grep -v 127.0.0.1

# または
ipconfig getifaddr en0
```

#### 3. URLを共有

```
http://[あなたのIPアドレス]:8501
```

例：`http://192.168.1.100:8501`

## 📦 方法3: GitHubでコードを共有

### メリット
- ✅ コードを公開できる
- ✅ 他の人が自分のパソコンで実行できる
- ✅ バージョン管理ができる

### デメリット
- ⚠️ 各人が環境をセットアップする必要がある

### 手順

1. GitHubにリポジトリを作成
2. コードをプッシュ
3. READMEに実行手順を記載
4. リポジトリのURLを共有

### READMEに記載する内容例

```markdown
# メルカリスクレイピングアプリ

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/あなたのユーザー名/リポジトリ名.git
cd リポジトリ名
```

2. 依存パッケージをインストール
```bash
pip install -r requirements_streamlit.txt
playwright install chromium
```

3. アプリを起動
```bash
streamlit run streamlit_app.py
```

4. ブラウザで http://localhost:8501 にアクセス
```

## 🔒 セキュリティに関する注意事項

### ⚠️ 重要な注意点

1. **スクレイピングの利用規約**
   - メルカリの利用規約を確認してください
   - 過度なリクエストは避けてください

2. **個人情報の保護**
   - アプリに個人情報を含めないでください
   - APIキーなどは環境変数で管理してください

3. **アクセス制限**
   - 公開する場合は、必要に応じて認証を追加してください
   - Streamlitの認証機能を使用できます

## 🆘 トラブルシューティング

### Streamlit CloudでPlaywrightが動かない場合

`requirements.txt`に以下を追加：

```
playwright>=1.40.0
```

デプロイ時に自動的にブラウザがインストールされます。

### ローカルネットワークで接続できない場合

1. ファイアウォールの設定を確認
2. 同じWi-Fiネットワークに接続されているか確認
3. IPアドレスが正しいか確認
