# メルカリスクレイピング実行コマンド

## ✅ Playwrightブラウザのインストール完了

ブラウザは `~/playwright-browsers/` に正常にインストールされました。

## 🚀 スクレイピング実行方法

**Cursorのターミナルで以下のコマンドを実行してください：**

```bash
cd /Users/shimadaeiji/Documents/Cursor/e2e
export PLAYWRIGHT_BROWSERS_PATH=~/playwright-browsers
python3 mercari/scrape.py
```

このコマンドで：
1. 環境変数を設定してブラウザのパスを指定
2. メルカリのポケモンカード検索結果ページにアクセス
3. 商品リンクを取得
4. 2件の商品情報を取得
5. CSVファイルに保存

## 📋 実行結果の確認

実行が完了すると、以下のファイルに結果が保存されます：

```
mercari/output/mercari_items.csv
```

## 🔍 トラブルシューティング

### 問題: ブラウザが見つからないエラーが発生する場合

**解決方法：環境変数を設定**

```bash
export PLAYWRIGHT_BROWSERS_PATH=~/playwright-browsers
python3 mercari/scrape.py
```

### 問題: タイムアウトエラーが発生する場合

**解決方法：待機時間を延長**

`mercari/scraper.py` の `wait_time` パラメータを増やしてください。
